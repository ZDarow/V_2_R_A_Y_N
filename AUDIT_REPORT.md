## 1. Общая характеристика проекта

Проект **v2rayN Russia Setup** — автоматизированный инструмент для настройки v2rayN/Xray в условиях интернет-блокировок на территории РФ. Предоставляет одноcommandный установщик, два режима роутинга, авто-обновление geoip/geosite правил, kill-switch, системный прокси, мобильный деплой на Android и всестороннюю диагностику.

**Тип проекта:** DevOps-инструментарий (bash + JSON + systemd)  
**Архитектура:** Монолитный набор shell-скриптов с общей библиотекой  
**Целевые платформы:** Linux (x86_64, aarch64), Android (v2rayNG)
