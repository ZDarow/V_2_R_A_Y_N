📱 v2rayNG

Настройка для РФ — загрузка конфигов с телефона

📂 Скачайте файлы

Нажмите на каждый файл — он скачается в память телефона.

routing-russia.json ~2 KB config-template-xray.json ~4 KB only_blocked.json мобильный ~2 KB
geoip.dat правила IP ~5 MB geosite.dat правила доменов ~8 MB

📋 Инструкция

1
Скачайте файлы выше — нажмите на каждый, чтобы сохранить в память телефона.
2
Перенесите скачанные файлы в папку
Android/data/com.v2ray.ang/files/assets/
(через любой файловый менеджер — Google Files, CX File Explorer и т.п.)
3
Откройте v2rayNG → Настройки → Настройки роутинга →
включите «Пользовательский файл роутинга» → выберите routing-russia.json (всё через VPN) или only_blocked.json (только заблокированное).
4
Добавьте подписку — нажмите + в v2rayNG → «Импорт по URL» → вставьте ссылку из раздела ниже.
5
Включите отпечаток сертификата в настройках подписки (долгое нажатие → карандаш → «Отпечаток сертификата»). allowInsecure удалён!

⚠️ allowInsecure

Xray-core отключает allowInsecure с 1 августа 2026.
v2rayNG 2.2.3 полностью удалил эту опцию. Вместо неё — только отпечаток SHA256 сертификата.

В настройках каждой подписки (долгое нажатие → карандаш) включите «Отпечаток сертификата» (pinnedPeerCertSha256). Если fingerprint не указан — скопируйте из конфига v2rayN или получите через openssl s_client ...
