📋 Инструкция

1
Скачайте файлы выше — нажмите на каждый, чтобы сохранить в память телефона.
2
Перенесите файлы в папку
Android/data/com.v2ray.ang/files/assets/
(через любой файловый менеджер — Google Files, CX File Explorer и т.п.)
3
Импортируйте правила роутинга
🔥 Достаточно нажать кнопку выше «📋 Copy JSON», затем:
v2rayNG → ≡ (меню) → Маршрутизация → ⋮ → Импорт из буфера обмена → OK

Альтернатива — из файла:
Скачайте файл выше → переместите в assets/ →
v2rayNG → Маршрутизация → ⋮ → Импорт правил из файла → выберите файл.

4
Смените доменную стратегию на IPOnDemand
(ВАЖНО! Без этого geoip:ru может не работать)
Маршрутизация → поле «Доменная стратегия» → выберите IPOnDemand (рекомендуется) или IPIfNonMatch. Не используйте «AsIs» — он отключает geoip.
5
Добавьте подписку одним нажатием

Нажмите на ссылку-кнопку ниже → v2rayNG откроется и предложит импорт.
Или нажмите + в v2rayNG → Import from URL → вставьте URL.

6
Включите отпечаток сертификата в настройках подписки (долгое нажатие → карандаш → «Отпечаток сертификата»). allowInsecure удалён!

📋 Формат правил

Формат правил основан на обсуждении v2rayNG Discussion #4761 (29 комментариев).

Поддерживаемые поля: domain, ip, protocol, network, port, outboundTag, enabled, remarks, looked.

Импорт через буфер обмена или файл — единственный поддерживаемый способ (deep link v2rayng://routing-rules не реализован, отклонён разработчиком).

⚠️ allowInsecure

Xray-core отключает allowInsecure с 1 августа 2026.
v2rayNG 2.2.3 полностью удалил эту опцию. Вместо неё — только отпечаток SHA256 сертификата.

В настройках каждой подписки (долгое нажатие → карандаш) включите «Отпечаток сертификата» (pinnedPeerCertSha256). Если fingerprint не указан — скопируйте из конфига v2rayN или получите через openssl s_client ...
